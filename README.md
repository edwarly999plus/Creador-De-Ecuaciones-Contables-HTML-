ContabilityMaker Interactive Panel 
Una herramienta interactiva desarrollada en HTML5, Tailwind CSS y Chart.js diseñada para visualizar el impacto de las transacciones financieras en la ecuación contable ($Activos = Pasivos + Capital$).El sistema permite a los usuarios navegar por una línea de tiempo de transacciones, observando en tiempo real cómo se desglosan los activos y cómo se mantiene el equilibrio contable mediante gráficos dinámicos y una tabla de análisis detallada.
